# 卡卡聚合支付系统 - KKPay

## 安装与配置

1. 将程序源码压缩包解压至网站根目录
2. 进入网站根目录
3. 执行命令 `php start.php start -d` 启动程序

> 程序基于Webman v2.1开发，推荐使用`PHP8.5`及以上版本，最低支持`PHP8.5`。
> 
> 程序采用MySQL数据库进行数据存储，推荐使用`MySQL9.0`及以上版本，最低支持`MySQL8.0`。

> 本程序需要`PDO_Mysql`+`Redis`+`BC Math`+`Event`组件，请检查是否安装并开启/运行。
